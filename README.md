# my-to-do

Simple TODO server wiritten in Sprig Boot, H2 in memory database

## Features
  - Users, who own projects, Project has events - short time events, aka "pomodoro"
  - some sample data - two users 
  - H2 console enabled
  - simple end-to-end tests
